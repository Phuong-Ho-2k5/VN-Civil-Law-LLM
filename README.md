# VN-Civil-Law-LLM
Hệ thống tư vấn Luật Dân sự Việt Nam sử dụng công nghệ RAG và Fine-tuning LLM.

## 🚀 Tính năng
- Trình diễn giao diện Chatbot hiện đại (HTML/CSS).
- Truy xuất kiến thức pháp luật chính xác từ Bộ luật Dân sự 2015 (RAG).
- Phản hồi thông minh nhờ model đã được Fine-tune.

## 🛠 Công nghệ sử dụng
- **AI:** Python, Unsloth, LangChain, ChromaDB.
- **Web:** FastAPI, HTML5, CSS3, JavaScript.
